# Plan: Add Favicon, SEO Meta Tags, and OG Image

## Overview
Add proper branding, SEO optimization, and social sharing support for karaoke0.

**App Name:** karaoke0  
**Tagline:** Spotify-like lyrics player for any song  
**URL:** https://karaoke0.vercel.app

---

## Step 1: Create Favicon and App Icons

Create simple text-based icons using SVG (can be converted to PNG/ICO).

### Files to Create in `public/`:
| File | Size | Purpose |
|------|------|---------|
| `favicon.svg` | 32x32 | Modern browsers (scalable) |
| `favicon.ico` | 32x32 | Legacy browser fallback |
| `apple-touch-icon.png` | 180x180 | iOS home screen |
| `icon-192.png` | 192x192 | PWA manifest |
| `icon-512.png` | 512x512 | PWA manifest |

### Design:
Simple "K0" or music note with dark background (#09090b zinc-950) and white text.

## Step 3: Update Meta Tags in `src/routes/__root.tsx`

Update the `head()` function:

```typescript
head: () => ({
  meta: [
    { charSet: "utf-8" },
    { name: "viewport", content: "width=device-width, initial-scale=1" },
    { title: "karaoke0 - Spotify-like lyrics player for any song" },
    { name: "description", content: "A beautiful lyrics player with word-by-word sync highlighting. Play any song with perfectly timed karaoke-style lyrics." },
    
    // Open Graph
    { property: "og:type", content: "website" },
    { property: "og:site_name", content: "karaoke0" },
    { property: "og:title", content: "karaoke0 - Spotify-like lyrics player for any song" },
    { property: "og:description", content: "A beautiful lyrics player with word-by-word sync highlighting. Play any song with perfectly timed karaoke-style lyrics." },
    { property: "og:image", content: "https://karaoke0.vercel.app/og-image.png" },
    { property: "og:url", content: "https://karaoke0.vercel.app" },
    
    // Twitter Card
    { name: "twitter:card", content: "summary_large_image" },
    { name: "twitter:title", content: "karaoke0 - Spotify-like lyrics player for any song" },
    { name: "twitter:description", content: "A beautiful lyrics player with word-by-word sync highlighting." },
    { name: "twitter:image", content: "https://karaoke0.vercel.app/og-image.png" },
    
    // Additional SEO
    { name: "theme-color", content: "#09090b" },
    { name: "author", content: "crafter-station" },
  ],
  links: [
    { rel: "stylesheet", href: appCss },
    { rel: "icon", type: "image/svg+xml", href: "/favicon.svg" },
    { rel: "icon", type: "image/x-icon", href: "/favicon.ico" },
    { rel: "apple-touch-icon", sizes: "180x180", href: "/apple-touch-icon.png" },
    { rel: "manifest", href: "/manifest.json" },
  ],
}),
```

## Image Generation Approach

Since we need simple text-based icons, we'll:
1. Create an SVG favicon with "K0" text (scalable, works in modern browsers)
2. Use an online tool or script to generate PNG versions from the SVG design
3. For the OG image, create a simple HTML-based design and screenshot it, or use a tool

**Alternative:** Use a favicon generator service like realfavicongenerator.net with a base SVG.
