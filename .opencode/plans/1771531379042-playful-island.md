# Landing Page Redesign Plan

## Summary
Replace the current hardcoded lyrics player at `/` with a proper landing page that has a URL input for Suno songs and a featured songs section.

## File to Modify
**`src/routes/index.tsx`** - Complete rewrite (only file that changes)

## Route Configuration

Add a **loader** that fetches metadata for 3 featured songs in parallel using the existing `getSongMetadata` server function from `@/server/lyrics`:

```typescript
const FEATURED_SONG_IDS = [
  "6cdb57d5-4c36-429e-a89c-7e12dd764271",
  "a440a785-5245-4e1d-91a8-abeb064430e8",
  "3f937066-957f-4572-b617-ffbae130e03d",
]

export const Route = createFileRoute("/")({
  loader: async () => {
    const metadata = await Promise.all(
      FEATURED_SONG_IDS.map((songId) => getSongMetadata({ data: { songId } }))
    )
    return {
      featuredSongs: FEATURED_SONG_IDS.map((id, i) => ({ id, ...metadata[i] })),
    }
  },
  component: LandingPage,
})
```

## Page Sections

### 1. Background Gradient (fixed, behind everything)
Reuse the existing radial-gradient pattern (purple/blue tones on zinc-950).

### 2. Hero Section
- Large "karaoke0" title with the "0" in a purple-to-blue gradient
- Subtitle: "Sing along to any Suno song with perfectly synced word-by-word lyrics."
- Entrance animation: fade-up (`opacity 0->1, y 20->0`)

### 3. URL Input Section
- Pill-shaped input container (`rounded-full`, `bg-zinc-900`, `border-zinc-800`)
- Placeholder: "Paste a Suno song URL..."
- White "Play" submit button (rounded-full, matching existing button style)
- On submit: extract UUID from URL via regex, navigate to `/song/$songId`
- Error message for invalid URLs shown below input
- Uses `useNavigate` from TanStack Router for programmatic navigation

### 4. Featured Songs Grid
- "Featured Songs" section heading (uppercase, tracking-widest, zinc-400)
- 3 cards in a responsive grid: `grid-cols-1 md:grid-cols-3`
- Each card uses `Link` to `/song/$songId` (enables preload on hover)
- Card contents: album cover image (aspect-square), song title, artist name
- Hover effect: image scale-up, card bg brightens
- Staggered entrance animations (0.1s delay between each card)

### 5. Footer
- Minimal: "Built with Suno + ElevenLabs"
- Low contrast (zinc-600), fades in last

## URL Parsing Logic
Extract UUID via regex: `/[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}/i`
- Supports: `https://suno.com/song/{id}`, `https://www.suno.com/song/{id}`, or raw UUID
- If no match: show inline error "Please enter a valid Suno song URL"
- Submit disabled when input is empty
